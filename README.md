# aws-webdeploy-example

This repo is an example of using the [aws-webdeploy](https://github.com/dsandor/aws-webdeploy) node module to deploy a static website.

# usage

Install dependecies.

```
yarn install
```

Deploy to AWS.

```
yarn deploy
```
